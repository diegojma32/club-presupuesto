# Sistema Presupuestario — Club Deportivo

Dashboard interactivo para gestión de presupuestos de clubes deportivos.

## Cómo usar

1. Abrí la URL del sitio
2. Cargá el archivo `presupuesto_club_v4.xlsx`
3. El dashboard se actualiza automáticamente

## Tecnología

HTML + JavaScript puro. Sin servidor. Sin base de datos. El archivo Excel se procesa localmente en el navegador del usuario.
